JS9 Imexam: imexam using JS9 plugins

You can add browser-based imexam analysis to a JS9 Web page!

See Adding Local Analysis Tasks (Plugins) to JS9 for details about plugins ...

In this example, ImExam div elements are defined in the Web-page.

Load an image, create a region, and move it around to see the imexam windows update. You also can select one or more light imexam windows from the Analysis menu to see other cuts of the data.

  • CAS-A (Chandra)
  • CTB 109 (Einstein)
  • m13 (via SkyView)
In this example, no ImExam div element has been defined on the page. Imexam plugins therefore will be offered in the Analysis menu as light windows.

Load an image, choose one or more plugins, create a region, and move it around to see the imexam windows update.

  • CAS-A (Chandra)
  • CTB 109 (Einstein)
  • m13 (via SkyView)
Load an image, create a region, select one or more imexam windows from the Analysis menu. Then move the regions around to see the imexam windows update.
